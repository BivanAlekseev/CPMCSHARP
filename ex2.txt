Код:
int max = 500;
for (byte i = 0; i < max; i++)
{
    Console.WriteLine(i);
}
Переменная i типа byte имеет диапазон от 0 до 255, и если в цикле происходит увеличение переменной i до значений, превышающих 255, то происходит переполнение. Это вызовет бесконечный цикл, так как i снова станет 0 после достижения 255, а условие i < max всегда будет истинным.

Модернизированный код (byte -> int):
int max = 500;
for (int i = 0; i < max; i++)  // Заменили byte на int
{
    Console.WriteLine(i);
}
